##  Project Description
Online Book Store is a backend RESTful application built using Spring Boot.  
It allows users to browse books, manage cart items, and handle authentication with role-based authorization.

This project was developed as a hands-on practice to improve backend development skills using Java and Spring technologies.

---

##  Features

- User registration and login
- JWT-based authentication
- Role-based authorization (USER / ADMIN)
- CRUD operations for books
- Shopping cart management
- Order processing
- API documentation with Swagger
- local host 8083

---

##  Technologies Used

- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- JPA / Hibernate
- PostgreSQL
- Gradle
- Swagger (OpenAPI)
- Git & GitHub
